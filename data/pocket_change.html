These files contain data on the Markov chain for a "big spender's" wallet state in four currencies considered in my paper with Lara Pudwell. The set of wallet states and the transition matrix are available. Each transition matrix is available in two formats — as a list of the nonzero entries and their positions (which can be passed to SparseArray) and in its full form as a list of lists. This data was computed using the package PocketChange.

U.S. currency: 6720 states sparse array rules (15 MB) transition matrix (144 MB)
pennyless purchaser: 213 states sparse array rules transition matrix
quarter hoarder: 4125 states sparse array rules (8 MB) transition matrix (55 MB)
Shallit currency: 4238 states sparse array rules (10 MB) transition matrix (58 MB)
