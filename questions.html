To the best of my knowledge, the following conjectures and questions are open.

prime-generating recurrences

  • Conjecture. Let n0 ≥ 1 and s0 ≥ 1. Let s(n0) = s0 and s(n) = s(n − 1) + gcd(n, s(n − 1)) for n ≥ n0 + 1. Then there exists N such that s(n) − s(n − 1) is either 1 or prime for all n ≥ N.
    reference: A natural prime-generating recurrence

  • Conjecture. Let s(1) = 7, and for n ≥ 8 let s(n) = s(n − 1) + gcd(n, s(n − 1)). Then every odd prime occurs in the sequence s(n) − s(n − 1).
    reference: A natural prime-generating recurrence

  • Conjecture (due to Benoit Cloitre). Let s(1) = 1, and for n ≥ 2 let s(n) = s(n − 1) + lcm(n, s(n − 1)). Then s(n)/s(n − 1) − 1 is either 1 or prime for all n ≥ 2.
    reference: A natural prime-generating recurrence

arithmetic properties of combinatorial sequences

  • Let R(α) be the number of residues modulo 2α attained by some Catalan number. Does R(α)/2α tend to 0 as α gets large?
    reference: Automatic congruences for diagonals of rational functions

  • Let C(n) be the nth Catalan number. Do there exist integers α ≥ 1 and r such that C(n) ≢ r mod 3α for all n ≥ 0?
    reference: Automatic congruences for diagonals of rational functions

  • Conjecture. Let M(n) be the nth Motzkin number. For all n ≥ 0, M(n) is not divisible by 312.
    reference: Automatic congruences for diagonals of rational functions

  • Let M(n) be the nth Motzkin number. Are there infinitely many primes p such that M(n) is not divisible by p2 for all n ≥ 0?
    reference: Automatic congruences for diagonals of rational functions

  • Let M(n) be the nth Motzkin number. Is the 3-adic valuation ν3(M(n)) unbounded? If so, is this sequence 3-regular?
    reference: Automatic congruences for diagonals of rational functions

  • Let A(n) be the nth Apéry number. Are there infinitely many primes p such that A(n) is not divisible by p for all n ≥ 0?
    reference: Automatic congruences for diagonals of rational functions

pattern avoidance in binary trees

  • Conjecture. Let s and t be n-leaf binary trees. Then s and t are avoiding-equivalent if and only if they are enumerating-equivalent.
    One direction is clear.

    reference: Pattern avoidance in binary trees

  • Conjecture. Let s and t be n-leaf binary trees. Then s and t are avoiding-equivalent if and only if there is a sequence of top-down replacement bijections, bottom-up replacement bijectoins, and left-right reflections that produces a bijection from binary trees avoiding s to binary trees avoiding t.
    reference: Pattern avoidance in binary trees
